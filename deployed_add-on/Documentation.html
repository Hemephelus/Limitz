Descriptive Statistics

Returns a summary of a dataset with information like Mean, Median, Standard Deviation, Skewness and so on.

Sample Usage

DescriptiveStatistics(B2:G42,1) or DescriptiveStatistics(B2:G42,A2)

Syntax

DescriptiveStatistics(data,[header])

data -(Numbers):

The range of cells to perform the calculations on.

header - (Boolean): [OPTIONAL]

To confirm if there is a header row type 1. If omitted, the value is guessed based on the content of data.

Multiple Linear Regression

Returns the coefficients of independent variables in the data set.

Sample Usage

MultipleLinearRegression(B2:G42,1) or MultipleLinearRegression(B2:G42,A2)

Syntax

MultipleLinearRegression(data,col,[header])

data -(Numbers):

The range of cells to perform the calculations on.

col - (Number)

The column the would be considers to be the dependent variable
